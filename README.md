# Chuks Kitchen

Chuks Kitchen is a modern food ordering web application that will allow customers to register, browse meals, place
orders, and track activity online. Users can also interact with a clean, responsive interface built with React and Tailwind CSS.

The application is designed using a mobile-first approach to ensure a smooth browsing experience across all screen sizes.


## Features

- Fully responsive design (Mobile-first)
- Browse popular food items
- View detailed food pages
- Dynamic routing with React Router


## Built With

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router



## Installation & Setup

1. Clone the repository:


  - git clone https://github.com/Bebe-dev/Chuks-Kitchen


2. Navigate into the project directory:


  - cd chuks-kitchen


3. Install dependencies:


  - npm install


4. Start the development server:


  - npm run dev


  - The app will run at: http://localhost:5173
